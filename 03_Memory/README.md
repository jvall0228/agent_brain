# Memory

Episodic memory — records of experience and extracted learnings.

## Structure

| Directory | Purpose | Template |
|-----------|---------|----------|
| `interactions/` | Logs of significant interactions | [[09_Templates/template-interaction-log]] |
| `reflections/` | Post-interaction analysis | [[09_Templates/template-reflection]] |
| `lessons/` | Extracted, reusable learnings | [[09_Templates/template-lesson]] |

## Lifecycle

1. **Interaction occurs** — Log it in `interactions/` if notable
2. **Reflect** — Write analysis in `reflections/`
3. **Extract** — Pull reusable lessons to `lessons/`
4. **Promote** — Validated patterns go to `06_Knowledge/patterns/`

Not every interaction warrants the full cycle. Log only what's notable.
