# Templates

Reusable note templates for structured content creation.

## Available Templates

| Template | Use Case | Destination |
|----------|----------|-------------|
| [[template-interaction-log]] | Record of a significant interaction | `03_Memory/interactions/` |
| [[template-reflection]] | Post-interaction analysis | `03_Memory/reflections/` |
| [[template-lesson]] | Extracted, reusable learning | `03_Memory/lessons/` |
| [[template-decision-record]] | Decision with reasoning | `04_Goals/` or `06_Knowledge/` |
| [[template-goal]] | Active objective | `04_Goals/` |
| [[template-domain]] | Area of competence | `05_Domains/` |
| [[template-knowledge]] | Reference knowledge | `06_Knowledge/facts/` |
| [[template-pattern]] | Recognized pattern or heuristic | `06_Knowledge/patterns/` |
| [[template-procedure]] | Step-by-step workflow | `06_Knowledge/procedures/` |

## Usage

Copy the template content, replace `{{PLACEHOLDERS}}` with actual values, and save to the indicated destination directory.
